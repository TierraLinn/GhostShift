# Install GhostShift on a phone

The fastest mobile test is the PWA version of GhostShift.

## Android

1. Make sure the GhostShift server is running on your computer.
2. Connect your phone to the same Wi-Fi network.
3. Find your computer IP address with:

```powershell
ipconfig
```

4. On Android Chrome, open:

`http://YOUR-COMPUTER-IP:8787`

5. Tap the browser menu.
6. Tap Add to Home screen or Install app.

## iPhone / iPad

1. Make sure the GhostShift server is running on your computer.
2. Connect your iPhone/iPad to the same Wi-Fi network.
3. Open Safari and go to:

`http://YOUR-COMPUTER-IP:8787`

4. Tap Share.
5. Tap Add to Home Screen.

## Important

The PWA is the mobile dashboard and setup app. Automatic skip behavior inside native YouTube/Facebook apps is not available from the PWA. Mobile skip assistance depends on browser extension support and store policy.
